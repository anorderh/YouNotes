A Chrome extension for writing & saving notes inside the YouTube player.

View the [Chrome Web store page](https://chromewebstore.google.com/detail/younotes/gehcckelgmjikeplalgonkhdojacadmg) here.

## Demo

[Youtube Video](https://www.youtube.com/watch?v=ekAdnSXZCKY&feature=youtu.be)

## Techstack

- **TypeScript** — Strongly typed extension logic and maintainable codebase  
- **HTML / CSS** — Lightweight, responsive UI embedded directly into YouTube  
- **Webpack** — Bundling and build pipeline for Chrome extension deployment  
- **Chrome Extension APIs** — Content scripts, storage, and browser integration  
- **Tiptap (Rich Text Editor)** — Enables headings, bold, lists, images, and structured note-taking  

## Core Capabilities

![younotes](https://github.com/user-attachments/assets/bdddefee-c42d-490c-b1f3-3a5a37241eb2)

- **In-Player Rich Text Notes**  
  Write fully formatted notes directly inside the YouTube video player (headings, bold, lists, etc.)

- **Timestamped & Visual Context**  
  Enhance notes with timestamps and embedded screenshots, allowing you to revisit exact moments with rich visual context  

- **Persistent Video-Linked Notes**  
  Notes automatically save and reload whenever you revisit a video  

- **Search & Organization**  
  Quickly find notes via the extension popup using search, sorting, and filtering  

## Use Cases

1. Take structured notes while watching tutorials, lectures, or technical content  
2. Revisit important video segments instantly with timestamped and visual references  
3. Build a centralized knowledge base tied directly to video content  
4. Reinforce learning through repeated review (spaced repetition-style workflows)
